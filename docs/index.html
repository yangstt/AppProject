部署步骤

方法一：用户/组织站点

  1. 创建名为 username.github.io 的仓库
  2. 在仓库根目录放置 index.html 文件
  3. 进入 Settings > Pages
  4. 选择 Deploy from a branch 并指定分支
  5. 网站将通过 https://username.github.io 访问

方法二：项目站点

  1. 在现有仓库中创建 docs 文件夹
  2. 将网站文件放入 docs 文件夹
  3. 进入 Settings > Pages
  4. 选择 Deploy from a branch 并指定 docs 文件夹
  5. 网站将通过 https://username.github.io/repository 访问
